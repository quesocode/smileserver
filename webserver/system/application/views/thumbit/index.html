Thumb.it

A thumbnailing web service by Swamped

Thumbnail

Resize, Crop a source image into a specified pixel dimension on the fly

URL Syntax

img/[width]x[height]/?src=[image source url]

Example

img/250x200/?src=http://farm4.static.flickr.com/3003/3066857751_13e70f6606.jpg

Thumbnail Dimensions

Display pixel dimensions of thumbnail

URL Syntax

img/[width]x[height]/

Example

img/250x250/

Thumbnail Dimensions Custom Colors

Display pixel dimensions of thumbnail

URL Syntax

img/[width]x[height]/&bg=[Hex color for bg]&color=[Hex color for font]

Example

img/250x200/

Thumbnail with Text

Overlay pixel dimensions on custom thumbnail

Example

img/280x330/filename.jpg?src=[image source url]&text=PEACE&fontsize=38&color=FF0000&font=arial_black

URL Syntax

img/[width]x[height]/?src=[image source url]&text=[url encoded text here]

Thumbnail with Text, Custom Colors, Box, and Courier New Bold Font

Overlay pixel dimensions on custom thumbnail

Example

img/250x250/filename.jpg?src=[image source url]&text=Example%202&bg=000000&fontsize=16&color=e3ea1c&font=courier_new_bold

URL Syntax

img/[width]x[height]/[image name (caching)]/?src=[image source url]&q=[quality (1-100)]&text=[url encoded text here]&bg=[Hex color for box around text]&fontsize=[font size in pixels]&color=[Hex color for font]&font=[font name]

Thumbnail with Text, Box, Custom Font, and Custom Position

Overlay pixel dimensions on custom thumbnail

Example

img/250x150/filename.jpg?src=[image source url]&q=100&text=Example%203&bg=FFFFFF&fontsize=16&color=FF0000&font=courier_new&x=left&y=bottom

URL Syntax

img/[width]x[height]/[image name (caching)]/?src=[image source url]&q=[quality (1-100)]&text=[url encoded text here]&bg=[Hex color for box around text]&fontsize=[font size in pixels]&color=[Hex color for font]&font=[font name]
